1. Type \ to begin typing a command.

2. Click a suggestion to autofill.

3. The glossary shows a list of available commands.

4. Click in the glossary to automatically insert a symbol.

5. ^ controls superscript. Available characters are [0-9], '+', '=', '(', ')'.

6. ^{ } allows multiple superscript replacement inside the brackets. Same symbol set as above.

7. _ controls subscript. Available characters are [0-9], '+', '=', '(', ')'.

8. _{ } allows multiple subscript replacement inside the brackets. Same symbol set as above.
